1 Project Setup

All work in ProSutra is organized into projects. You must create or select a project before you can run any jobs.

  1. Go to the Dashboard.
  2. To create a new project, enter a unique name in the "Create New Project" card and click the button.
  3. To work on an existing project, simply click its name in the "Select Existing Project" list.
  4. The currently active project is always displayed at the top of the dashboard.

2 Protein Modeling

Once a project is active, navigate to the Modeling page to generate 3D structures from a FASTA sequence.

  1. Provide a unique Job Name for your modeling run.
  2. Paste your protein sequence in the FASTA Sequence box.
  3. Select your desired modeling tools:
    • Modeller: For local, template-based homologous modeling. Requires Modeller to be installed.
    • Robetta: For remote, ab initio (de novo) modeling. Requires credentials to be set in the Settings page.
  4. Click Run Modeling. For Robetta jobs, you can monitor the queue and download results from the "Robetta Job Management" section.

3 Structure Validation

After creating models, use the Validation page to assess their quality.

  1. Choose your PDB file source:
    • Project Model: Select one or more models from the currently active project.
    • Upload PDB: Drag and drop or click to upload PDB files from your computer.
  2. Select the validation tools you wish to run from the checklist.
  3. Click Run Validation.

4 Viewing Reports

After a validation job completes, you will be automatically taken to the Reports page.

  1. The page will display a set of cards for each validation tool that was run.
  2. Each card shows a status (e.g., completed, failed) and a summary of the results.
  3. For tools that generate plots or have detailed text output, you can view them directly in the card.
  4. Use the "Show Log" expander to see the raw output from a tool, which is useful for debugging.
