Validation Tools

Once a protein model is generated, it is crucial to assess its quality. ProSutra integrates several validation tools to help you evaluate your models.

DSSP (Define Secondary Structure of Proteins)

Description: DSSP is a program that calculates the secondary structure of a protein from its atomic coordinates.

Significance: It provides a standard definition of secondary structure elements such as alpha-helices and beta-sheets, which is essential for understanding the protein's fold.

FreeSASA

Description: FreeSASA calculates the solvent accessible surface area (SASA) of a protein.

Significance: SASA is useful for analyzing protein stability and interactions with other molecules. It can help identify exposed and buried residues.

MolProbity

Description: MolProbity is a comprehensive tool for evaluating the quality of a protein structure model.

Significance: It provides a variety of scores, including clashscore (steric clashes), Ramachandran plot analysis (phi/psi angles), and rotamer analysis (side-chain conformations). These scores help identify potential problems in the model's geometry.

ProSA (Protein Structure Analysis)

Description: ProSA is a tool for checking the quality of a protein structure model based on its energy profile.

Significance: It provides a Z-score that indicates the overall quality of the model compared to a database of known protein structures. A lower Z-score generally indicates a better model.

QMEAN (Qualitative Model Energy Analysis)

Description: QMEAN is a composite scoring function for estimating the quality of a protein structure model.

Significance: It provides a QMEAN score that ranges from 0 to 1, with higher scores indicating better models. It also provides individual scores for different geometric features.

VoroMQA (Voronoi-based Model Quality Assessment)

Description: VoroMQA is a tool for assessing the quality of a protein model using Voronoi tessellation.

Significance: It provides a global quality score and a local quality score for each residue, which can help identify problematic regions in the model.
